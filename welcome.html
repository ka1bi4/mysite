Meet CodeLobster

Welcome to CodeLobster IDE help!

CodeLobster is an Integrated Environment for web developers. CodeLobster has rich functionality for editing HTML,CSS,JavaScript and PHP, working with VCS (version control), SQL. Main feature - plugins for CMS and frameworks: CakePHP, Codelgniter, Drupal, Joomla, Symphony, Twig, WordPress, Yii, Node.js, JQuery, AngularJS, BackboneJS.

CodeLobster brings you the following advanced features:

  • Intelligent Editor: for PHP, HTML, CSS, JavaScript, and XML, which includes syntax highlighting, documentation lookup, and refactorings.

  • Project and Code Navigation: specialized project views, file structure views and quick navigation between files, classes, methods and usages.

  • Error-Free Coding: on-the-fly code analysis, error highlighting and quick fixes.

  • VCS Integrations: out-of-the-box support for Git, Subversion, Tortoise SVN, Mercurial with changelists and merge.

  • FTP Sync: update server using FTP or SFTP.

  • SQL support: coding assistance, SQL console and Database browser.
